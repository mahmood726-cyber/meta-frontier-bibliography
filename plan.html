Project Plan

Meta-Frontier Annotated Bibliography

This project packages the recent methods scan as a browser-native annotated bibliography: a filterable reading map for frontier meta-analysis methods from 2024 to early 2026.

Standalone HTML app Curated paper records Theme filters WebR integrity checks

Scope

  • Track the most relevant recent methods papers from statistics and methodology journals.
  • Annotate each paper with what is new, why it matters, and how mature it looks.
  • Make the bibliography usable as a reading queue rather than a static citation list.

Entry Schema

  • ID
  • Theme
  • Journal and year
  • Maturity band
  • Short annotation
  • Source link

Planned Themes

Small-sample inference Sparse data and rare events Selective reporting and p-hacking Bayesian bias adjustment Transportability IPD survival NMA Individualized treatment effects Distributional robustness HTA multi-indication

Core Views

  • Headline metrics for papers, years, and themes
  • Filter bar for theme, year, and maturity band
  • Annotated paper cards with direct source links
  • Starter-reading list for fast orientation

WebR Validation Plan

  • Check total paper count and unique IDs in base R.
  • Check minimum and maximum publication year.
  • Check theme counts.
  • Check the newest-first top paper IDs.

This validates the integrity of the embedded bibliography dataset and the page ordering logic.

Build Order

  • 1. Freeze the bibliography set and write the annotations.
  • 2. Build the filter and card UI in a single HTML file.
  • 3. Add the starter-reading section.
  • 4. Add the WebR integrity panel and script preview.
  • 5. Place the project in C:\Projects with index.html and this plan.
